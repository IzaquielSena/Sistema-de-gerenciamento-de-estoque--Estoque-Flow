# Sistema-de-gerenciamento-de-estoque--Estoque-Flow
Estoque Flow: Sistema local (PHP/MySQL/XAMPP) criado para a Ximac Pesca. Eliminou processos manuais, furos de mercadoria e falta de visibilidade que o cliente enfrentava, automatizando o controle de inventário e gerando relatórios dinâmicos em PDF via Dompdf para tomadas de decisão rápidas e precisas.
